# Cross-Platform Weekday Script

A lightweight, cross-platform Bash script to instantly display the current weekday name and its corresponding number.

## Features
- Works natively on **Linux** and **macOS**.
- Supports **Windows** via Git Bash, WSL, or MSYS2.
- Displays the full weekday name (e.g., *Saturday*) and day number (1 for Monday, 7 for Sunday).

## Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com
   cd cross-platform-weekday
   ```

2. Make the script executable:
   ```bash
   chmod +x showeekday.sh
   ```

3. Run the script:
   ```bash
   ./showeekday.sh
   ```
